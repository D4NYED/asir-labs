# Ubuntu VM Deployment (VirtualBox)

## Objetivo
Desplegar una máquina virtual Ubuntu 24.04 LTS utilizando VirtualBox para crear un entorno Linux de laboratorio.

## Entorno
- VirtualBox (Type 2 Hypervisor)
- Ubuntu 24.04 LTS ISO
- 4GB RAM
- 20GB Disk

## Pasos realizados
- Descarga de ISO oficial
- Creación de máquina virtual Linux
- Asignación de recursos (RAM/CPU/Storage)
- Instalación mínima del sistema
- Instalación de drivers y software adicional
- Primer arranque y configuración de usuario
- Actualización del sistema
