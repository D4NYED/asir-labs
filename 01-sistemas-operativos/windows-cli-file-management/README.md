# Windows CLI File Management (CMD)

## Objetivo
Administrar archivos y carpetas desde la línea de comandos de Windows (CMD) para realizar tareas de creación, copia, renombrado y eliminación de forma rápida y automatizable.

## Tareas realizadas
- Creación de directorios (md)
- Navegación por rutas (cd)
- Creación de archivos vacíos (copy nul, echo)
- Copias de seguridad de archivos (copy)
- Eliminación de archivos (del)
- Copia recursiva de directorios (xcopy /E /I)
- Renombrado de carpetas (ren)
- Eliminación de directorios no vacíos (rmdir /S)

## Enfoque técnico
Uso de CLI para automatización, administración masiva y mayor control frente a la interfaz gráfica.

## Resultado
Gestión completa del sistema de archivos únicamente mediante comandos.
