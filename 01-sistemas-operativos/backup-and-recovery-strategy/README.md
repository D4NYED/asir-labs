# Backup and Recovery Strategy (EaseUS Todo Backup)

## Objetivo
Implementar un sistema de copias de seguridad automatizadas para proteger datos críticos y permitir su recuperación ante fallos o pérdida de información.

## Tareas realizadas
- Instalación del software de backup
- Configuración de copias incrementales diarias
- Copias completas mensuales
- Programación automática de tareas
- Verificación de integridad de respaldos
- Restauración de archivos
- Uso de múltiples destinos (HDD externo, NAS, USB, FTP, nube)

## Enfoque técnico
- Automatización de backups
- Estrategia de recuperación (disaster recovery)
- Redundancia de almacenamiento
- Minimización de pérdida de datos

## Resultado
Sistema de copias fiable, automatizado y con recuperación rápida de información.
