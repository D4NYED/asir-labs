# Windows Server VM Deployment

## Objetivo
Desplegar máquinas virtuales Windows Server 2016 y 2022 en VirtualBox para crear un entorno de laboratorio de administración de sistemas.

## Entorno
- VirtualBox
- Windows Server 2016 / 2022
- 4–8 GB RAM
- 50 GB Disk
- NAT + Internal Network

## Configuración realizada
- Creación de VM
- Asignación de CPU/RAM/Storage
- Montaje de ISO
- Instalación Server Core y Desktop Experience
- Configuración de red
- Contraseñas seguras
- Primer arranque y verificación

## Casos de uso del laboratorio
- Active Directory
- Servicios de red
- Administración remota
- Pruebas de seguridad
