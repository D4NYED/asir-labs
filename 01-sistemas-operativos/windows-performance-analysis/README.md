# Windows Performance Monitoring & Bottleneck Analysis

## Objetivo
Diagnosticar problemas de rendimiento en Windows utilizando Performance Monitor (perfmon) y recopiladores de datos para identificar cuellos de botella en CPU, memoria y disco.

## Tecnologías
- Performance Monitor (perfmon)
- Data Collector Sets
- Resource Monitor
- Event Viewer

## Tareas Realizadas
- Creación de Data Collector Sets personalizados
- Monitorización continua de CPU, RAM y disco
- Generación de logs históricos (.blg)
- Análisis de picos de carga y saturación
- Identificación de posibles cuellos de botella

## Métricas Clave
Ver `counters-cheatsheet.md`

## Resultado
Capacidad para detectar degradación de rendimiento y tomar decisiones basadas en datos (capacity planning / tuning).
