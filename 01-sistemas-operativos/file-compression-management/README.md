# Gestión de software de compresión (7-Zip)

## Objetivo
Instalar, configurar y desinstalar un software de compresión en Windows para optimizar el envío y almacenamiento de archivos.

## Tareas realizadas
- Evaluación de alternativas (7-Zip vs WinRAR/WinZip)
- Instalación del software
- Integración con menú contextual
- Compresión y descompresión de archivos
- Configuración avanzada (LZMA2, nivel Ultra)
- Cifrado AES-256 con contraseña
- Desinstalación limpia del sistema

## Enfoque técnico
- Gestión de aplicaciones en endpoint
- Optimización de almacenamiento
- Protección de archivos mediante cifrado

## Resultado
Software instalado y operativo, archivos comprimidos con alta tasa de compresión y cifrado seguro.
